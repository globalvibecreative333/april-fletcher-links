Where should this tool live?

This shapes everything about how it's built and how fast you can open it mid-session.

A

Single HTML file

One self-contained file you open in any browser. No install, no server, no dependencies. Double-click to launch. Works offline.

B

Local web app (React / Next.js)

Runs with npm run dev. Easier to extend over time, but requires Node.js installed and a terminal step to launch.

C

Claude-powered (AI evaluates leads)

The Lead Evaluator uses Claude API to score snippets. Requires an API key and internet. Best accuracy, but adds setup and cost.
