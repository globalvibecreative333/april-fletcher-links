Database Schema

Four tables in Supabase

PROJECTS
id uuid pk
name text
category text // enum in app
status text // active|paused|waiting|done
last_worked_on date
notes text
created_at timestamptz
OPEN_LOOPS
id uuid pk
project_id uuid → projects.id
type text // follow-up|blocker
description text
resolved bool (default false)
due_date date nullable
created_at timestamptz
CHAT_MESSAGES NEW
id uuid pk
role text // user|assistant
content text
created_at timestamptz
Trim on app load: DELETE WHERE created_at < now() - interval '6 months'
USER_SETTINGS NEW
id int pk (always 1)
idle_amber_days int (default 5)
idle_red_days int (default 7)
theme text (default 'dark')
updated_at timestamptz
Single row. Seed with defaults on init.

Category enum (app-level, fixed set)
